# HH Parser and Vacancy Classifier

Этот проект представляет собой набор скриптов для парсинга вакансий с HeadHunter (HH.ru) и их последующей классификации.

## Статус проекта

**ВНИМАНИЕ:** Проект находится на стадии активной разработки и еще не завершен. Функционал может изменяться, и некоторые части могут быть неоптимальными или содержать ошибки.

## Структура проекта

*   `all city/`
    *   `parse_country.py`: Скрипт для сбора вакансий по определенному ключевому слову со всей России (или другого указанного региона) и сохранения их в CSV файл.
*   `filte city/`
    *   `filter_city.py`: Модуль, содержащий функции для получения вакансий по конкретному городу.
    *   `save_csv.py`: Скрипт, который использует `filter_city.py` для парсинга вакансий по указанному городу и ключевому слову, с последующим сохранением в CSV. Этот файл не включает описание вакансий из сниппета.
    *   `save_csv_2.py`: Расширенная версия `save_csv.py`, которая также извлекает и сохраняет описание вакансий из сниппета HH.
    *   `vacancies_набережные челны.csv`: Пример CSV файла с вакансиями (будет игнорироваться при выгрузке в Git).
*   `main.py`: Основной файл для демонстрации базового функционала парсинга вакансий с HH.
*   `use model/`
    *   `check.py`: Скрипт для проверки доступности PyTorch и GPU.
    *   `classification_model.py`: Содержит базовую реализацию для загрузки модели `facebook/bart-large-mnli` из библиотеки `transformers` для классификации текста.
    *   `test.py`: Скрипт, использующий модель классификации для категорийного анализа вакансий из CSV файла, добавляя категории и скоры в новый CSV.

## Зависимости

Для работы проекта требуются следующие библиотеки:

```
requests
torch
transformers
```

Вы можете установить их с помощью `pip`:

```bash
pip install -r requirements.txt
```
